Hesap Makinesi

Proje Yapılışı

    Evet, şimdi öğrendiklerimizi uygulamaya geldik. Bir önceki sayfada öğrendiklerimizi test etmiştik. Proje yaparken üç tane aşamamız vardır.

Düşünme

    Düşünme aşamasında, projemizin ne olacağı ve neler içereceği planlanır.
    Projemiz, konsol üzerinden çalışan bir hesap makinesi olacak ve dört işlemi desteklemeli. Ayrıca listeleri kullanarak geçmiş özelliği de olmalı. Çıkış yapılana kadar çalışmalı ve döngülerle kodlanmalı. Tamam, düşünme bu kadar olsun. Önemli bilgileri not edelim.

Gereksinimleri Karşılama

    Şimdilik dış kütüphaneye ihtiyacımız yok. Bu yüzden bu kısmı atlıyoruz. Ancak ilerleyen sayfalarda bu kısmı using ile kullanacağız.

Yapılış

    Artık düşündüklerimizi ve planladıklarımızı pratiğe dökme vakti!
namespace SMCS
{
    internal class Program
    {
        static void Main(string[] args)
        {
            List gecmis = new List();

            while (true) // Ana döngü
            {
                Console.Clear(); // Konsolu temizler

                Console.WriteLine("1: Toplama");
                Console.WriteLine("2: Çıkarma");
                Console.WriteLine("3: Çarpma");
                Console.WriteLine("4: Bölme");
                Console.WriteLine("5: Geçmişi Görüntüle");
                Console.WriteLine("0/Q: Çıkış");
                Console.Write("-->");

                string entry1 = Console.ReadLine();

                if (entry1.Equals("1")) // Toplama kısmı
                {
                }
                else if (entry1.Equals("2")) // Çıkarma kısmı
                {
                }
                else if (entry1.Equals("3")) // Çarpma kısmı
                {
                }
                else if (entry1.Equals("4")) // Bölme kısmı
                {
                }
                else if (entry1.Equals("5")) // Geçmişi görüntüleme
                {
                    Console.Clear(); // Konsolu temizler

                    foreach (string islem in gecmis) // İşlemleri yazdırma
                    {
                        Console.WriteLine(islem);
                    }

                    Console.Read();

                    Console.Clear(); // Konsolu temizler

                    continue; // Döngüyü başa sar
                }
                else if (entry1.Equals("0") || entry1.Equals("Q") || entry1.Equals("q")) // Çıkış
                {
                    break; // Hesap makinesini kapatma
                }
            }
        }
    }
}
    Proje geliştirirken kodları adım adım yorumlamak, daha sağlıklı projeler üretmemizi sağlar. Yazdığımız kodlarla programa seçimleri tanıttık. Şimdi toplama kısmını yazalım.
Console.Clear(); // Konsolu temizler

while (true) // Toplama döngüsü
{
    Console.Write("Birinci sayıyı giriniz: "); // Birinci sayıyı isteme

    string s1String = Console.ReadLine(); // Kullanıcıdan String türünde girdi alma
    int s1 = 0; // Birinci sayı

    try // Eğer kullanıcı sayı yerine string girerse, program hata verip çökebilir. Bunu önlemek için Try-Catch bloklarını kullandık
    {
        s1 = Convert.ToInt32(s1String); // Dönüştürme
    }
    catch (Exception e) // Dönüştürmede hata alınırsa
    {
        Console.WriteLine("Girdiğiniz veri sayı değildir!"); // Hata mesajı

        Console.Read();

        continue; // Döngüyü başa sarma
    }

    Console.Write("İkinci sayıyı giriniz: "); // İkinci sayıyı isteme

    string s2String = Console.ReadLine(); // Kullanıcıdan String girdisi isteme
    int s2 = 0; // İkinci sayı

    try // Eğer kullanıcı sayı yerine string girerse, program hata verip çökebilir. Bunu önlemek için Try-Catch bloklarını kullandık
    {
        s2 = Convert.ToInt32(s2String); // Dönüştürme
    }
    catch (Exception e) // Dönüştürmede hata alınırsa
    {
        Console.WriteLine("Girdiğiniz veri sayı değildir!"); // Hata mesajı

        Console.Read();

        continue; // Döngüyü başa sarma
    }

    string islem = "\n" + s1String + " + " + s2String + " = " + (s1 + s2); // İşlemin String hali

    Console.WriteLine(islem); // Konsolda gösterme

    gecmis.Add(islem); // Geçmişe ekler

    Console.Read();

    Console.Clear(); // Konsolu temizler

    break; // Toplama döngüsünü kırma
}
    Evet, hesap makineniz toplama işlemini yapabilir durumda! Peki diğer işlemler? İşte onun kodlarını yazmak size düşüyor! Yazdığımız komutları temel alarak devam edebilirsiniz.
