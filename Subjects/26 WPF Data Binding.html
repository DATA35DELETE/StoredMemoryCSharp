Data Binding

Data Binding Nedir?

    Data Binding türkçesiyle veri bağlama, GUI bileşenlerinin, Backend kısmında direk veriye bağlamamızı sağlayan çok önemli bir yöntemdir. WPF'in en olduğu kısım budur.

Modları

    Data Binding birden fazla modları vardır. Her biri farklı durumlarda kullanılır.

OneTime

    OneTime, bileşene bir kere verisini taşır.
<Grid>
    <TextBlock Text="{Binding KullaniciAdi}" 
                FontSize="20" 
                HorizontalAlignment="Center" 
                VerticalAlignment="Center"/>
</Grid>
    XAML tarafında bir Grid'in içine TextBox yerleştirdik. Şimdi onu Backen d'e kodlara bağlıyalım.
namespace storedMemoryWPF
{
    public partial class MainWindow : Window
    {
        public MainWindow()
        {
            InitializeComponent();
            this.DataContext = new KullaniciViewModel();
        }

        public class KullaniciViewModel
        {
            public string KullaniciAdi { get; set; } = "Stored Memory C#";
        }
    }
}

TwoWay

    TwoWay, hem veri değiştiğinde hemde GUI'daki veri değiştiğinde güncellenir.
<StackPanel Margin="20">
        <TextBox Text="{Binding KullaniciAdi, Mode=TwoWay, UpdateSourceTrigger=PropertyChanged}" 
                 FontSize="16" Margin="0,0,0,10"/>
        <TextBlock Text="{Binding KullaniciAdi}" FontSize="18" Foreground="RoyalBlue"/>
    </StackPanel>
    C#:
public partial class MainWindow : Window
{
    public MainWindow()
    {
        InitializeComponent();
        this.DataContext = new KullaniciViewModel(); // {Binding KullaniciAdi} diyebilmemiz için önemli
    }
}

public class KullaniciViewModel : INotifyPropertyChanged // Data Binding için Class
{
    private string _kullaniciAdi = "Data Delete"; // Ana veri
    public string KullaniciAdi // Değişken veri
    {
        get => _kullaniciAdi; // Bileşen bunu okur
        set
        {
            _kullaniciAdi = value; // Değeri günceller
            OnPropertyChanged(nameof(KullaniciAdi)); // Değiştiğini bileşene haber gider.
        }
    }

    public event PropertyChangedEventHandler PropertyChanged;
    protected void OnPropertyChanged(string propertyName)
    {
        PropertyChanged?.Invoke(this, new PropertyChangedEventArgs(propertyName));
    }
}
