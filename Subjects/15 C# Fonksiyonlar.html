Fonksiyonlar

Fonksiyon Nedir?

    Fonksiyon, birden fazla kod parçasını bir arada tutan yapıdır. Yani fonksiyonun içerisinde birden fazla kod bulunur ve tek bir komutla o kodlar çalıştırılır. Genellikle birden fazla yerde kullanılacak kod parçalarını tek bir yerde tutarak kod tasarrufu sağlayan yapılardır. Oluşturma formülü: erişimBelirteci static geriyeDeğerDöndürmeTürü fonksiyonAdı(parametreler) { Kodlar }. Çağırma formülü: fonksiyonAdı(parametre);

Void Fonksiyonlar

    Void fonksiyonlar, geriye değer döndürmeyen fonksiyonlardır. Yani bu fonksiyonlar çağrıldığında bir değer döndürmez.
public static void fonksiyon()
{
    // Kodlar
}

Return’lü Fonksiyonlar

    Return’lü fonksiyonlar, void fonksiyonların aksine geriye değer döndürür. "return" ifadesi ile değer döndürülür. Eğer return kullanılmazsa hata oluşur.
public static int fonksiyon()
{
    // Kodlar

    return 15; // Bu kısımda değer döndürülür
}

Parametreler

    Parametre, fonksiyonların parantezlerinde tanımlanan verilerdir ve fonksiyonlar arasında iletişimi sağlar. Fonksiyon oluşturulurken parantezlerine "veriTürü veriAdı" şeklinde tanımlanır.
public static void yazdir(string text)
{
    Console.WriteLine(text);
}

...

yazdir("Hello");
// Konsolda "Hello" yazısı görülecektir.
}
