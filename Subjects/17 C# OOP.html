OOP

OOP Nedir?

    OOP, yani Object Oriented Programming (Nesne Yönelimli Programlama), C# programlama dilinin temelini oluşturur. OOP, programlama dillerinde fazla kod yazmayı önlemek için belirli yöntemler kullanır.
    Dört prensibe dayanır:

Prensipler

Kapsülleme (Encapsulation)

    Verileri ve fonksiyonları dışarıdan yapılacak gereksiz işlemlere karşı korur. Böylece yapının iç işleyişi korunur.
public class Araba
{
    // Private field’lar (dışarıdan direkt erişilemez)
    private int hiz;
    private string model;

    // Public property’ler (kontrollü erişim için)
    public int Hiz
    {
        get { return hiz; }
        set
        {
            if (value >= 0 && value <= 300) // Hız kontrolü
                hiz = value;
            else
                throw new Exception("Hız 0-300 arasında olmalı!");
        }
    }

    public string Model
    {
        get { return model; }
        set { model = value; }
    }

    // Public metot
    public void Hizlan(int artis)
    {
        Hiz += artis;
        Console.WriteLine($"{Model} {Hiz} km/s hıza ulaştı!");
    }
}

public static void Main(string[] args)
{
    // Kullanımı
    Araba araba = new Araba();
    araba.Model = "Toyota";
    araba.Hizlan(50); // Toyota 50 km/s hıza ulaştı!
}

Kalıtım (Inheritance)

    Kalıtım, OOP prensiplerinde en önemli öğelerden biridir. Kalıtım, bir sınıftaki özellikleri başka bir sınıfa aktarmaktır.
    Örneğin, tüm canlıların ortak özellikleri beslenme ve solunum yapmalarıdır. Bu iki özelliği Canlilik adında bir sınıfa koyalım. Ardından Kedi adında bir sınıf oluşturalım. Kedilerin özelliklerini yazdıktan sonra : ile Canlilik sınıfını ekleyelim.
public class Canlilik // Canlıların iki ortak özelliği
{
    public void beslenme() // Beslenme fonksiyonu
    {
        Console.WriteLine("Canlı beslendi");
    }

    public void solunum() // Solunum fonksiyonu
    {
        Console.WriteLine("Canlı nefes alıp verdi");
    }
}

public class Kedi : Canlilik // Kedi canlısının, Canlilik sınıfındaki özelliklerini yükledik
{
    public string ismi = "Boncuk";
    public byte yasi = 2;
}

public static void Main(string[] args) // Ana fonksiyon
{
    Kedi canli = new Kedi(); // Kedimizi oluşturuyoruz

    canli.beslenme(); // Kedimiz Canlilik sınıfından gelen beslenme fonksiyonunu çalıştırıyor
    canli.solunum();  // Kedimiz Canlilik sınıfından gelen solunum fonksiyonunu çalıştırıyor

    Console.WriteLine("Canlının ismi: " + canli.ismi); // Kedinin ismi
    Console.WriteLine("Canlının yaşı: " + canli.yasi); // Kedinin yaşı
}
    Böylece kalıtım özelliğini kullanmış olduk.

Çok Biçimlilik (Polymorphism)

    Çok biçimlilik, aynı ada sahip fonksiyonların farklı işlemler yapabilmesini sağlayan prensiptir. Çoğunlukla fonksiyonları gruplandırabilmek için kullanılır.
public static void Main(string[] args)
{
    Console.WriteLine(selamVer());       // Teknik olarak aynı fonksiyonu çağırsalar da
    Console.WriteLine(selamVer("Data")); // Çok biçimlilik sayesinde aynı fonksiyon fakat farklı işlemler yapılabilir.
}

public static string selamVer()
{
    return "Selam";
}
public static string selamVer(string name)
{
    return "Selam, " + name;
}

Soyutlama (Abstraction)

    Soyutlama, içeriği olmayan ancak var olan fonksiyonlar oluşturmak için kullanılan bir prensiptir. Gerekli yerlerde gövdesiz, yani içeriği olmayan fonksiyonlar içerikle doldurularak kullanılır.
public interface IHareketEdebilir
{
    void HareketEt();
}

public class Araba : IHareketEdebilir
{
    public void HareketEt()
    {
        Console.WriteLine("Araba 4 tekerlekle hareket ediyor!");
    }
}

public class Ucak : IHareketEdebilir
{
    public void HareketEt()
    {
        Console.WriteLine("Uçak havada uçuyor!");
    }
}

public static void Main(string[] args)
{
    // Kullanımı
    IHareketEdebilir araba = new Araba();
    IHareketEdebilir ucak = new Ucak();
    araba.HareketEt(); // Araba 4 tekerlekle hareket ediyor!
    ucak.HareketEt(); // Uçak havada uçuyor!
}
