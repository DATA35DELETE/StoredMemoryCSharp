Karar Yapıları

Karar Yapıları Nedir?

    Karar yapıları, programlamada koşullara göre şekillenmeyi sağlayan yapılardır.
    Karar yapıları sayesinde koşullara göre farklı kodlar çalıştırabiliriz. Örneğin; bir banka sisteminde kullanıcı, ismini ve şifresini giriyor. Biz bu verilerle sistemleri kontrol ederek kullanıcının gerçekten var ise bir hoş geldin ekranı, yok ise hata mesajı verebiliriz.

If-Else If-Else

If

    If, bu yapının başıdır. İlk olarak onun koşullarına bakılır. Parantezlerindeki koşul true ise kod bloklarını çalıştırır.
if (koşul)
{
    // Koşul true ise burası çalışır
}

Else If

    Else if, bu yapının ara elemanıdır. Bir If’in altına konulmalıdır. If çalışmaz ve parantezlerindeki koşul true ise kod bloklarını çalıştırır.
if (koşul1)
{
    // Koşul1 true ise burası çalışır
}
else if (koşul2)
{
    // Koşul1 false ve Koşul2 true ise burası çalışır
}

Else

    Else, bu yapının son elemanıdır. Else if’in aksine bir If’in altına sadece bir tane Else konulabilir. If ve Else if’ler çalışmazsa kendi kod blokları çalışır.
if (koşul1)
{
    // Koşul1 true ise burası çalışır
}
else if (koşul2)
{
    // Koşul1 false ve Koşul2 true ise burası çalışır
}
else
{
    // Koşul1 ve Koşul2 false ise burası çalışır
}

Ternary Operatörü

    Ternary, minik bir If-Else bloğu gibi çalışır. Soru işareti (?)’nin sağ tarafındaki koşula göre, iki nokta (:)’nın sağ veya sol tarafı çalışır.
bool koşul = true;
bool koşul2 = false;

Console.WriteLine(koşul ? "Koşul true" : "Koşul false"); // Konsola “Koşul true” yazdırır
Console.WriteLine(koşul2 ? "Koşul2 true" : "Koşul2 false"); // Konsola “Koşul2 false” yazdırır

Switch-Case

    Switch-Case yapısı tek verileri sorgulamak için kullanılır. Mantık olarak If-Else If-Else yapısıyla aynıdır. Birlikte kullanılmalıdır.
string renk = "Kırmızı";

switch (renk) // Sorgulanacak veri
{
    case "Kırmızı": // Eğer renk değişkeni “Kırmızı” ise
        Console.WriteLine("Renk Kırmızı");
        break; // Break, Case’in sonlandığını belirtir. Yoksa diğer Case’lere de bakar

    case "Mavi": // Eğer renk değişkeni “Mavi” ise
        Console.WriteLine("Renk Mavi");
        break;

    case "Yeşil": // Eğer renk değişkeni “Yeşil” ise
        Console.WriteLine("Renk Yeşil");
        break;

    default: // Eğer Case’ler hiçbiri değilse, Else gibi çalışır
        Console.WriteLine("Bilinmeyen Renk");
        break;
}
