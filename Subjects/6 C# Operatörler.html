Operatörler

Operatör Nedir?

    Operatörler, basit matematik işlemleri veya mantıksal işlemler için kullanılan araçlardır.

Atama Operatörü

    Atama, bir verinin değerini başka bir değişkene aktarmasıdır. Atama işleminin operatörü eşittir (=) işaretidir.
int A = 10;
int B = 20;

A = B; // A’nın değeri artık 20 olur, B’nin değeri A’ya aktarılır.

Aritmetik Operatörler

    Aritmetik operatörler, matematiksel işlemler için kullanılan operatörlerdir.

Toplama (+)

    İki sayının toplamını alır.
int A = 10;
int B = 20;

int C = A + B; // C’nin değeri 30 olur

Çıkarma (-)

    İki sayının farkını alır.
int A = 20;
int B = 10;

int C = A - B; // C’nin değeri 10 olur

Çarpma (*)

    İki sayının çarpımını alır.
int A = 10;
int B = 20;

int C = A * B; // C’nin değeri 200 olur

Bölme (/)

    İki sayının bölümünü alır.
int A = 20;
int B = 10;

int C = A / B; // C’nin değeri 2 olur

Aritmetik Atama Operatörleri

    Bu operatörler, hem aritmetik işlem yapar hem de atama işlemini gerçekleştirir. Kullanımı: değer1 aritmetikOperatör= değer2

Karşılaştırma Operatörleri

    Karşılaştırma operatörleri, iki değişkenin değerini karşılaştırarak Boolean bir değer döndürür.

Eşittir (==)

    İki değerin eşit olup olmadığını kontrol eder. Eşitse True, değilse False döner.
int A = 10;
int B = 20;

bool C = A == B; // C’nin değeri False olur

Eşit Değildir (!=)

    İki değerin eşit olup olmadığını kontrol eder. Eşit değilse true, eşitse false döner.
int A = 10;
int B = 20;

bool C = A != B; // C’nin değeri true olur

Büyüktür (>)

    Soldaki değerin sağdaki değerden büyük olup olmadığını kontrol eder. Büyükse true, değilse false döner.
int A = 20;
int B = 10;

bool C = A > B; // C’nin değeri true olur

Küçüktür (<)

    Soldaki değerin sağdaki değerden küçük olup olmadığını kontrol eder. Küçükse true, değilse false döner.
int A = 10;
int B = 20;

bool C = A < B; // C’nin değeri true olur

Büyük Eşittir (>=)

    Soldaki değerin sağdaki değerden büyük veya eşit olup olmadığını kontrol eder. Büyük veya eşitse true, değilse false döner.
int A = 20;
int B = 10;

bool C = A >= B; // C’nin değeri true olur

Küçük Eşittir (<=)

    Soldaki değerin sağdaki değerden küçük veya eşit olup olmadığını kontrol eder. Küçük veya eşitse true, değilse false döner.
int A = 10;
int B = 20;

bool C = A <= B; // C’nin değeri true olur

Mantıksal Operatörler

    Mantıksal operatörler, birden fazla koşulu birleştirmek veya tersine çevirmek için kullanılır.

Ve (&&)

    İki koşulun da doğru (true) olması durumunda true döner. Eğer herhangi bir koşul false ise false döner.
bool A = true;
bool B = false;

bool C = A && B; // C’nin değeri false olur

Veya (||)

    İki koşuldan en az birinin doğru (true) olması durumunda true döner. Her iki koşul da false ise false döner.
bool A = true;
bool B = false;

bool C = A || B; // C’nin değeri true olur

Değil (!)

    Bir koşulun tersini alır. Koşul true ise false, false ise true döner.
bool A = true;
bool B = !A; // B’nin değeri false olur

Yorum Operatörü

    Yorum, programlamada yazılım dilinin dikkate almadığı ve yazılımcıların kodlarını diğer yazılımcılara açıklamasını sağlayan ifadelerdir. SMC# boyunca her kodu yorum operatörü ile açıklayacağım. Ayrıca “Temiz Kod” prensibinin temelini oluşturur.
namespace storedMemoryCSharp
{
    internal class Program
    {
        static void Main(string[] args)
        {
            Console.WriteLine("Hello, World!"); // Bu kod Console’a “Hello World” yazdırır
        }
    }
}
