Listeler

Liste Nedir?

    Liste, programlama dillerinde birden fazla veriyi tek bir yapıda toplamak, düzenlemek veya işlemek için kullanılır.
    Listelerin önemi, birden fazla veriyi sıralamak veya toplu şekilde işlemek gerektiğinde ortaya çıkar. Çok sayıda değişken tanımlamak yerine, tek bir veri yapısında birden fazla değer saklanabilir.

Projeye Dahil Etmek

    Bu sayfaya kadar kullandığımız yapılar, C#’ın içinde gömülüdür ve projemize ayrıca dahil etmemiz gerekmez. Ancak Listeler gibi yapılar, using ile projemize dahil edilir. Bu işlem, listeleri kullanmamızı sağlar.
using System.Collections.Generic;

List Kullanımı

    List, C#’ta liste yapılarını kullanmak için bir sınıftır. Yapısı: List<veriTürü>.

Metotları

namespace SMCS
{
    internal class Program
    {
        static void Main(string[] args)
        {
            List<string> list = new List<string>(); // String veri türünde Liste 

            list.Count(); // Count, listenin eleman sayısını verir. Integer türünde
            list.IndexOf("String"); // IndexOf, listede parantezlerindeki elemanın indeks numarasını verir

            list.Contains("String"); // Contains, listede parantezlerindeki elemanın olup olmadığını kontrol eder. Boolean türünde
            list.Equals(list); // Equals, listeyi parantezlerindeki listeyle karşılaştırır. Boolean türünde

            list.Add("String"); // Add, listeye eleman ekler
            list.Clear(); // Clear, listedeki tüm elemanları siler
            list.Remove("String"); // Remove, listede parantezlerindeki elemanı siler
            list.Reverse(); // Reverse, listeyi ters çevirir
            list.Sort(); // Sort, listeyi alfabetik olarak sıralar
        }
    }
}

Foreach

    Foreach, “Döngüler” konusunda gördüğümüz gibi For döngüsünün bir alt türüdür.
foreach (string str in list)
{
    Console.WriteLine(str);
}
    Bu kod, listedeki tüm elemanları sırasıyla yazdırır.
