Paralel Programlama

Paralel Programlama Nedir?

    Paralel programlama, işlemcilerin her çekirdeğinde farklı işlemleri gerçekleştirebilmek için kullanılan yöntemdir.
    Örneğin, bir pencere oluşturulurken programa ayrılan çekirdek bu işi yaparken, aynı anda veritabanında işlem yapamaz. Çünkü bilgisayarlarda her program için ayrı işlemci çekirdekleri tahsis edilir. Bazı durumlarda işlemlerimiz için tahsis edilen çekirdekler yetersiz kalır. Bu yüzden paralel programlama ile işlemciden yeni bir çekirdek tahsis edilir.

Task ve Async/Await

    Task, C# programlama dilinde modern bir paralel programlama sınıfıdır. Task ile fonksiyonlar paralel programlamaya uygun hale getirilir, aksi takdirde hata oluşabilir.
namespace storedMemoryCSharp
{
    internal class Program
    {
        static async Task Main(string[] args)
        {
            // Kodlar
        }
    }
}
    Böylece Main fonksiyonu paralel programlamaya hazır hale gelir. Aynı zamanda async ile farklı çekirdekte çalışacağı belirtilir.

Await

    await, Task’ta bir işlemin tamamlanmasını bekleyen anahtar kelimedir. await, program akışını belirtilen işlem tamamlanana kadar durdurur. İşlem tamamlandığında program akışı devam eder.
namespace storedMemoryCSharp
{
    internal class Program
    {
        static async Task Main(string[] args)
        {
            Console.WriteLine("Başlıyoruz...");
            await UzunSurenIslem();
            Console.WriteLine("Bitti!");
        }
        static async Task UzunSurenIslem()
        {
            Console.WriteLine("Uzun süren işlem başladı.");
            await Task.Delay(2000); // 2 saniye bekler (simüle edilmiş uzun işlem)
            Console.WriteLine("Uzun süren işlem tamamlandı.");
        }
    }
}
    Task.Delay() ile program belirtilen milisaniye kadar bekletilir.
