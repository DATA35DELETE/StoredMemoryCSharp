Hata Yönetimi

Hata Yönetimi Nedir?

    Hata yönetimi, programlama dillerinde bir hata oluştuğunda programın otomatik olarak kapanmasını önler. Kodda potansiyel hata üretebilecek komutları kontrol ederek programın çökmeden devam etmesini sağlar. Genellikle veritabanı işlemleri, dosya işlemleri ve kullanıcı girdisi alma gibi durumlarda potansiyel hataları önler.

Try-Catch

    Try-Catch, C# programlama dilinde hata yönetimini sağlayan bir yapıdır. try bloğu, potansiyel hatalı kodların yazıldığı kısım; catch bloğu ise hata oluştuğunda ve hatanın türüne bağlı olarak çalışan kodlardır. Hata çıkmazsa kod akışı devam eder.
try
{
    // Potansiyel hata çıkarabilecek kodlar
}
catch (Exception e) // Hata türüne bağlı catch
{
    // Hata çıkarsa çalışacak kodlar
}
// İstenildiği kadar catch eklenebilir.
    catch parantezlerindeki hata türü, hata mesajını da içerir. Bazı durumlarda kendi hata mesajlarımızı oluşturmamız gerekir. Bu durumlarda throw ile kendi hata mesajlarımızı oluştururuz.
throw new Exception("Hata Mesajı");
