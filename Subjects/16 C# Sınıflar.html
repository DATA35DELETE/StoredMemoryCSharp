Sınıflar

Sınıf Nedir?

    Sınıflar, C# dilinde değişkenleri veya fonksiyonları gruplandırabildiğimiz yapılardır.
    Sınıflar, tıpkı gerçek hayattaki okul sınıflarına benzer. Bu sınıflar içinde, örneğin öğrenci sayısını byte, öğrencilerin çantalarındaki suyun litresini float ve öğretmenin ismini string olarak ifade edebiliriz. Öğretmenin ders anlatması gibi eylemler, sınıf içindeki fonksiyonlara örnektir.
public class Araba // Sınıfımız
{
    public string Marka; // Marka değişkeni
    public int Hiz; // Hız değişkeni

    public void Hizlan() // Fonksiyonu
    {
        Hiz += 10;
    }
}

Nesne Oluşturma

    Nesne oluşturma, sınıfların bir kopyasını oluşturup üzerinden kullanmaktır.
    Örneğin, bir fabrikanın ürün üretmesi gibi. Fabrika, orijinal ürünü bire bir üreterek mağazalara gönderir, ancak orijinal ürün asla mağazalara verilmez.

Yapılışı

    Nesne oluşturmak için new anahtar kelimesi kullanılır. Gerisi değişken tanımlama gibidir.
class sinif // Sınıf
{
    public int ogrenciSayisi = 28; // Öğrenci sayısı
    public float ogrencilerinSuMiktarlari = 17f; // Öğrencilerin su miktarı (Litre)

    public string ogretmenIsmi = "Tilbe Bozkurt"; // Öğretmen ismi

    public void dersAnlat() // Ders anlatma fonksiyonu
    {
        Console.WriteLine("Evet gençler, bugün C# programlama dilinde değişkenlerin türlerini göreceğiz."); // Ders anlatımı
    }
    
    public int yoklamaAl(int i) // Sınıf yoklaması
    {
        return ogrenciSayisi - i;
    }
}

public static void Main(string[] args) // Ana fonksiyon
{
    sinif atp10 = new sinif(); // 10 ATP isimli sınıfı oluştur
    
    Console.WriteLine("Ders öğretmeni: " + atp10.ogretmenIsmi); // Ders öğretmeni
    Console.WriteLine(atp10.yoklamaAl(2)); // Öğretmen yoklamayı aldı
    
    atp10.dersAnlat(); // Ders anlattı
}

Static

    static, nesne oluşturmadan sınıfların içeriğinin kullanılmasını sağlar.
    static, sınıfta önemli bilgiler varsa bir yerde yapılan değişikliğin diğer kısımlara yansımasını sağlar.
    Kullanmak için sınıfın içindeki bileşenlere static ifadesi eklenir.
class sinif // Sınıf
{
    public static int ogrenciSayisi = 28; // Öğrenci sayısı
    public static float ogrencilerinSuMiktarlari = 17f; // Öğrencilerin su miktarı (Litre)

    public static string ogretmenIsmi = "Tilbe Bozkurt"; // Öğretmen ismi

    public static void dersAnlat() // Ders anlatma fonksiyonu
    {
        Console.WriteLine("Evet gençler, bugün C# programlama dilinde değişkenlerin türlerini göreceğiz."); // Ders anlatımı
    }

    public static int yoklamaAl(int i) // Sınıf yoklaması
    {
        return ogrenciSayisi - i;
    }
}

public static void Main(string[] args) // Ana fonksiyon
{
    Console.WriteLine("Ders öğretmeni: " + sinif.ogretmenIsmi); // Ders öğretmeni
    Console.WriteLine(sinif.yoklamaAl(2)); // Öğretmen yoklamayı aldı

    sinif.dersAnlat(); // Ders anlattı
}

Using

    using ile projelerimize gerekli kütüphaneler eklenir. Örneğin, dosya işlemleri için using System.IO; eklenir.
using System.IO;

Namespace

    namespace, sınıflarımızı sınıflandırmak için kullanılır. Proje oluşturulurken kullanılan isim, ana namespace olarak tanımlanır.
using System;

namespace SMCS
{
    // Kodlar
}
