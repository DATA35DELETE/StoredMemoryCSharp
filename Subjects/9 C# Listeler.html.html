Listeler

Liste Nedir?

    Liste, programalama dillerinde birden fazla verileri tek bir yapıda toplamak, düzenlemek veya işlemek için kullanılır.
    Listelerin önemi; Bazı zamanlarda birden fazla verileri sıralamak veya toplu bir şekilde işlemek istediğimiz zaman listeleri kullanırız. Sayıca fazla değişken tanımlamak yerine tek bir veri yapıda birden fazla değer olabilir.

Projeye Dahil Etmek

    Bu sayfaya kadar kullandığımız yapılar, C#'ın içerisinde gömülü olup, herhangi birini projemize dahil etmemiz gerekmez. Fakat Listeler gibi yapıları Using yardımı ile projemize dahil ederiz. Bu işlem bize listeleri kullanmamızı sağlar.
using System.Collection.Generic;

List Kullanımı

    List, C#'da liste yapılarını kullanabilmek için sınıftır. Yapısı; List<veriTürü>.

Metodları

namespace SMCS
{
    internal class Program
    {
        static void Main(string[] args)
        {
            List list = new List(); // String veri türünde List 

            list.Count(); // Count, listenin eleman sayısını verir. Integer türünde
            list.IndexOf("String"); // IndexOf, listede de parantezlerindeki elemanın Index numarası

            list.Contains("String"); // Contains, listede paratezlerindeki eleman var mı diye kontrol eder. Boolean türünde
            list.Equals(list); // Equals, listeyi parantezlerindeki listeyle karşılaştırır. Boolean 

            list.Add("String"); // Add, listeye eleman ekler
            list.Clear(); // Clear, listedeki tüm elemanları
            list.Remove("String"); // Remove, listede parantezlerindeki elemanı siler
            list.Reverse(); // Reverse, listeyi ters çevirir
            list.Sort(); // Sort, listeyi alfabetik olarak sıralar
        }
    }
}

Foreach

    Foreach, "Döngüler" konusunda gördüğümüz gibi For döngüsünün bir alt türüdür.
foreach (string str in list)
{
    Console.WriteLine(str);
}
    Bu kod, list'deki tüm elemanları sırasıyla sıralar.
