Veri

Veri Nedir?

    Veri, programlamada bilgileri geçici olarak RAM’de, kalıcı olarak SSD/HDD’de saklanan yapılardır. Diğer adı değişkenlerdir.
    Verilerin önemi, örneğin, bir hesap giriş-çıkış sisteminde kullanıcıdan isim alınır. Programda kullanıcının girdiği isim, değişkene kaydedilerek ihtiyaç durumunda çağrılır. Eğer böyle bir yapı olmazsa, programda sürekli kullanıcı ismini girmek zorunda kalınırdı.

Sayısal Veri Türleri

sByte

    sByte, C# programlama dilinde en küçük aralığa sahip sayı türüdür. -128 ila 127 sayılarının aralığında değer alabilir.
sbyte sekizBitlikBirDeğer = 8;

Short

    Short, sByte türünden daha büyük bir sayı aralığına sahiptir. -32,768 ila 32,767 sayılarının aralığında değer alabilir.
short onAltıBitlikDeğer = -1000;

Integer

    Integer (int), C# programlama dilinde en çok kullanılan sayı türüdür. -2,147,483,648 ila 2,147,483,647 sayılarının aralığında değer alabilir.
int otuzİkiBitlikDeğer = 2000000;

Long

    Long, Integer sayı veri türünün yetersiz kaldığı zamanlarda kullanılan sayı türüdür. -9,223,372,036,854,775,808 ila 9,223,372,036,854,775,807 sayılarının aralığında değer alabilir.
long altmışDörtBitlikDeğer = 9000000000;

Float

    Float, ondalıklı sayılar için kullanılan veri türüdür. Yaklaşık olarak 7 basamağa kadar hassasiyet sağlar. 32 bitlik bir veri türüdür.
float ondalikliSayi = 5.75f;
    Float veri türünde ondalıklı sayılar tanımlarken, sayının sonuna ‘f’ harfi eklenmelidir. Aksi takdirde derleyici hata verecektir.

Double

    Double, Float veri türünden daha hassas ondalıklı sayılar için kullanılan veri türüdür. Yaklaşık olarak 15-16 basamağa kadar hassasiyet sağlar. 64 bitlik bir veri türüdür.
double dahaHassasOndalikliSayi = 19.99d;
    Double veri türünde ondalıklı sayılar tanımlarken, sayının sonuna ‘d’ harfi eklenmelidir. Aksi takdirde derleyici hata verecektir.

Decimal

    Decimal, Double veri türünden daha hassas ondalıklı sayılar için kullanılan veri türüdür. Yaklaşık olarak 28-29 basamağa kadar hassasiyet sağlar. 128 bitlik bir veri türüdür.
decimal enHassasOndalikliSayi = 199.99m;
    Decimal veri türünde ondalıklı sayılar tanımlarken, sayının sonuna ‘m’ harfi eklenmelidir. Aksi takdirde derleyici hata verecektir.

Yazısal Veri Türleri

Char

    Char, tek bir karakteri saklamak için kullanılan veri türüdür. Tek tırnak işaretleri (‘ ’) arasında tanımlanır veya ASCII tablosundaki ondalıklı sayıların değerine tekabül eden karakteri tutar. 16 bitlik bir veri türüdür.
char büyükAHarfi = 'A';
büyükAHarfi = 65; // ASCII tablosunda A harfinin sayısal karşılığı 65’tir.

String

    String, birden fazla karakteri saklamak için kullanılan veri türüdür. Çift tırnak işaretleri (“ ”) arasında tanımlanır. 16 bitlik karakterlerden oluşan bir dizi (array) yapısıdır.
string metin = "Merhaba Dünya";

Boolean Veri Türü

    Boolean, programlama dillerinde True veya False değerlerini alabilen 1 bitlik veri türüdür.
bool dogruMu = true;
bool yanlisMi = false;

Null

    İstisnalar hariç, C#’ta referans türleri null değerini alabilir. Null, boş veya yok anlamına gelen bir kelimedir.
string metin = null;
    Null’un faydası: Değeri henüz belli olmayan referans türleri için null kullanılır. Eğer değişkenin değeri tanımlanmazsa program hata verebilir. Null sayesinde bu sorun ortadan kaldırılır.
    Zararı ise, null olan bir değişkenle işlem yapılmaya çalışılırsa NullReferenceException hatası alınır. Sebebi: Null boşluk olduğu için, herhangi bir işlemde kullanılamaz ve program orada kendini durdurur.

Erişim Belirteçleri

Public

    Public, C# programlama dilinde yapıları veya değişkenleri, diğer sınıfların veya fonksiyonların o veriye ulaşabilmesini sağlayan bir erişim belirtecidir.
public int herkeseAçıkSayı = 1881;

Protected

    Protected, sadece içerisinde bulunduğu sınıfa ve ondan kalıtılmış sınıflara erişilebilirlik sağlar.
protected string diğerSınıflarcaKorunmuş = "Doğum Yılı";

Private

    Private, C#’ta sadece tanımlandığı sınıf içerisinde kullanım erişimi sağlar.
private string özelŞifre = "ruhi123";

Erişim Belirteçleri Tablosu

Erişim Belirteci Tanım Erişim Alanı
Public Herkese Açık Her yerden erişilebilir.

Private Özel Sadece tanımlandığı sınıf içerisinden erişilebilir.

Sabitlik

Const

    Const, C# programlama dilinde verileri sabit değerle tanımlamayı sağlar. Ardından tanımlanan değişkenin değeri bir daha asla değiştirilemez.
const string buÜlkeninKurucusu = "Mustafa Kemal Atatürk";

Verileri Birbirlerine Dönüştürme

    Verileri birbirine dönüştürme, elimizdeki tür ihtiyaç duyulan türden farklı olduğunda gerekir. Bu yüzden Convert ile veriler dönüştürülür.
string sayi = "1000";
int sayiInt = Convert.ToInt32(sayi); // string türündeki sayi
// değişkeni int türüne dönüştürülüp sayiInt değişkenine atandı.
