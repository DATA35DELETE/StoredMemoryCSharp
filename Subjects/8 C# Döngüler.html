Döngüler

Döngü Nedir?

    Döngüler, bir olayı veya kod parçalarını belli bir koşula veya belirli bir miktar kadar çalıştıran yapılardır.
    Döngülerin önemi, örneğin, konsola yüz kere “Hello” yazdırmamız gerektiğinde, `Console.WriteLine("Hello")`’ı yüz kere yazmak yerine döngüler sayesinde bu işi üç-dört satırda halledebiliriz. Yani kod tasarrufu sağlar.

While

    While, C# programlama dilinde parantezlerindeki koşul `true` olduğu sürece kod bloğunu çalıştırır.

While döngülerde sonsuz döngü riski vardır. Eğer kodunuzda sonsuz döngü varsa program hatalarla karşılaşabilir!

while (koşul)
{
    // Kodlar
}

For

    For, C# programlama dilinde parantezlerindeki belirtilmiş koşula bağlı olarak, belirlenen miktar kadar döner. While’ın aksine bu döngüde sonsuz döngü riski yoktur.
for (int i = 0; i < döneceğiMiktar; i++)
{
    // Kodlar
}

Foreach

    Foreach, For döngüsünün alt türüdür. Verdiğimiz listeleri eleman sayısı kadar döner ve her adımda bir sonraki elemana geçer. Bu, “Listeler” sayfasında daha detaylı ele alınacak.

Break

    Break, Switch-Case yapısında gördüğümüz gibi çalıştırıldığı an döngüyü sonlandırır ve kod akışı devam eder.
while (koşul)
{
    // Kodlar
    if (koşul)
    {
        break; // Döngüyü sonlandırır.
    }
}

Continue

    Continue, döngülerde Break’in aksine döngüyü durdurmaz, mevcut adımı atlar ve bir sonraki adıma geçer.
while (koşul)
{
    // Kodlar
    if (koşul)
    {
        continue; // Mevcut adımı atlar ve bir sonraki adıma geçer.
    }
}
