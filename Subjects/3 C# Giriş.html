Yazılım Nedir?

    Yazılım, bir bilgisayardaki donanımlara ne yapacaklarını elektriksel olarak ifade eden kod veya kod bütünlüğüdür.
    Günlük hayatta yazılım, özellikle günümüzde, neredeyse tüm teknolojik aletlerde kullanılmaktadır. Örneğin, şu an bu SMC#’ü okumanızı sağlayan telefon, tablet veya bilgisayar gibi cihazlarda, siz görmeseniz de arkada binlerce kod çalışır ve ekrana bu bilgiler gönderilerek sizin bu yazıları okumanızı sağlar. Eğer yazılım diye bir kavram olmasaydı, bu cihazlar ve günümüzde çoğu alet işlevsiz kalırdı. Ayrıca donanım dediğimiz, yazılımın var olmasını sağlayan devrelerdir. Eğer yazılım var olmasaydı yer kaplamaktan başka bir işe yaramazdı. Günümüzde yapay zeka gibi fenomenler sayesinde yazılıma ilgi daha da artmıştır.

Yazılım Dilleri

    Yazılım dilleri, yazılımı insan dillerine benzeterek oluşturulan özel dillerdir. Aynı İngilizce, Türkçe veya İspanyolca gibi, yazılım dilleri arasında Python, Java, Ruby gibi yüzlerce dil ortaya çıkmıştır. Bunlardan en popüler olanlar Python ve Java’dır.
    Yazılım dillerinin türleri vardır. Bu, bir derecelendirme meselesidir. Genel olarak Low-Level ve High-Level olarak iki türe ayrılır. Bu derecelendirme, sanılanın aksine kolaylık değil, İngilizce’ye yakınlık meselesidir. Low-Level, İngilizce’ye uzak; High-Level ise yakın olduğunu ifade eder. Örneğin, Low-Level olan C++’taki bir kod, High-Level olan Python’daki bir koddan daha zor anlaşılır.

C# Nedir?

    Geldik SMC#’ün konusu olan C# yazılım diline! Bu dil, 2000 yılında Microsoft tarafından .NET teknolojisi için geliştirilmiş, C familyasına ait bir programlama dilidir. Bu dil, High-Level olarak derecelendirilir ve öğrenimi, aynı familyadan olan C ve C++’ın aksine daha kolaydır. Java ile benzerliği sayesinde, Java bilenler için C# bir ayda öğrenilebilir bir programlama dilidir.

İlk Program

    IDE kurulumundan önce bu komutları görmenizi isterim:
using System; // Kullanılan kütüphaneler

namespace storedMemoryCSharp // Proje adı
{
    class main // Sınıf adı
    {
        static void Main(string[] args) // Ana fonksiyon
        {
            Console.WriteLine("Hello World");

            Console.Write("Hello ");
            Console.Write("World");
        }
    }
}

Console.WriteLine ve Console.Write Farkı

    Özünde aynı işi yapsalar da, Console.WriteLine satır koyarak, Console.Write ise satır bırakmayarak konsola yazdırır.
