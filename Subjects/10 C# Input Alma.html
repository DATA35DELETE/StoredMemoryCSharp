Input Alma

Input Nedir?

    Input, programlarda kullanıcının verdiği verileri ifade etmek için kullanılır.
    Bu sayfaya kadar statik programlar yazdık; yani yazılımcının belirlediği değişkenlerle işlemler yaptık. Dinamik, yani kullanıcıyla etkileşimli programlar için input çok önemlidir. Günümüzde birkaç uygulama hariç tüm uygulamalar dinamik yazılır.

Console.ReadLine

    C# programlama dilinde Console.ReadLine() komutu ile konsol üzerinden kullanıcıdan yazı girdisi alınır. Bu komut, kullanıcının yazdığı ifadeyi döndürür. Bu nedenle, bir string veri türünde değişkene atanır, yoksa veri boşa gider.
namespace storedMemoryCSharp
{
    internal class Program
    {
        static void Main(string[] args)
        {
            Console.Write("İsminiz: "); // Kullanıcıya bilgi vermek için
            string name = Console.ReadLine(); // Kullanıcıdan veri alma

            Console.WriteLine("Hoş geldiniz " + name); // Kullanıcıya merhaba deme
        }
    }
}
    Console.ReadLine(), yalnızca string veri türü döndürür. Örneğin, int veri türüne ihtiyaç duyulduğunda veri türü dönüştürülür.
namespace storedMemoryCSharp
{
    internal class Program
    {
        static void Main(string[] args)
        {
            Console.Write("Birinci sayıyı giriniz: ");
            int sayi1 = Convert.ToInt32(Console.ReadLine()); // String’i int’e çevirme

            Console.Write("İkinci sayıyı giriniz: ");
            int sayi2 = Convert.ToInt32(Console.ReadLine()); // String’i int’e çevirme

            int toplam = sayi1 + sayi2; // Toplama işlemi
            Console.WriteLine("Toplam: " + toplam); // Sonucu gösterme
        }
    }
}
